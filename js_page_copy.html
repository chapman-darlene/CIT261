Variables

To understand JavaScript, we need to understand how JavaScript stores information. The simplest way is through creating variables. A variable in JavaScript is easily created by placing the keyword \"var\" in front of a unique name, known as an identifier. For example: var text = \"This is some text\". Variables must follow simple rules, such as:

  • Variables may include letters (a-z, A-Z) or numbers (0-9). It only allows the symbols $ (dollar sign) and _ (underscore)
  • No other characters, spaces or punctuation is allowed.
  • The first character can only be a letter, the $ sign or _
  • Names are case-sensitive. For instance: Text, text and TEXT are not the same identifiers.
  • There are no set limits on variable names, but they should reflect the item to be defined.
  • Camel-case variable names are strongly suggested in defining a variable name, such as: var = firstName.

Do-while loops are similar to loops, except it will iterate once through the code before any test is made. The code will look similar to this:

  • text += (count + " times 7 is " + count * 7 + "<br>")

Conditional Statements

Conditional statements alter the program's flow. It provides the programmer to ask questions and receive responses, sometimes with input from the user. There are three types of of non-looping conditionals: the if statement, the switch statement and the ? operator.

The If Statement:

  • var a = 100;
  • var b = 10;
  • if(a === b){
  • document.write("a is equal to b");

The If...Else Statement

We can add an else statement to this code that allows for an alternative response

  • var a = 100;
  • var b = 10;
  • if(a === b){
  • document.write("a is equal to b");
  • else {
  • document.write("a is not equal to be")
  • }

The ? Operator

We can simplify the ternary operation by using the question mark (?) and the colon (:) symbols: where the ? is equivalent to the if statement, and the colon is the else statement.

  • size = a <=5 ? "short" : "long";

Functions

The primary purpose of JavaScript is to provide dynamic content to a website. This is made possible primarily through the use of functions. JavaScript is a C-based program, high-level programming language, so it is relatively easy to learn in comparison to many lower-level languages. Understanding how to create simple to complex functions is an essential skill for any JavaScript programmer.
Functions are used to separate and organize code into smaller, reusable blocks. A simple function is created as follows:

  • <script>
  • function sample(a, b) {
  • return a*b
  • }
  • </script>
